# Эталон — премиальный уход за домом

Полноценный прототип сайта для премиального клининг-сервиса в Новосибирске.

## Что внутри

- `frontend/` — React + Vite лендинг с премиальным визуальным стилем, квизом и формой заявки.
- `backend/` — Node.js + Express API для приема заявок.
- Frontend отправляет заявки на `/api/leads`.
- Backend может сохранять заявки локально в `backend/data/leads.json` и отправлять уведомления в Telegram при наличии переменных окружения.

## Быстрый запуск

```bash
npm run install:all
npm run dev
```

Frontend: http://localhost:5173  
Backend: http://localhost:4000

## Telegram-уведомления

Создайте файл `backend/.env` по примеру `backend/.env.example`.

Если Telegram не настроен, заявки всё равно сохраняются в `backend/data/leads.json`.

## Перед запуском

1. Подставить реальный телефон, WhatsApp, Telegram.
2. Добавить реальные фото сотрудников, интерьеров и деталей чистоты.
3. Подключить CRM/почту/Telegram вместо демо-хранилища.
4. Добавить политику конфиденциальности и согласие на обработку персональных данных.
